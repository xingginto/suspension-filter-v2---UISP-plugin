# Client Suspension Filter V2.0

A comprehensive UISP plugin for filtering and managing clients based on suspension status, suspension dates, archive status, and client tags.

## Features

- **Suspension Status Filtering**: Filter clients by suspended service status
- **Date-based Filtering**: Filter clients by suspension date ranges
- **Archive Status Management**: Include or exclude archived clients
- **Tag-based Filtering**: Filter clients using custom tags
- **Comprehensive Reporting**: Generate detailed client suspension reports
- **Clean Interface**: User-friendly admin interface integrated into UCRM

## Requirements

- **UNMS Version**: 1.0.0 or higher
- **UCRM Version**: Compatible with UCRM plugin system

## Installation

1. Download the plugin files
2. Upload to your UCRM installation's plugin directory
3. Install through UCRM plugin management interface
4. Access via Reports menu

## Usage

1. Navigate to **Reports > Client Suspension Filter** in UCRM
2. Apply filters based on your requirements:
   - Suspension status (suspended/active)
   - Suspension date range
   - Archive status
   - Client tags
3. View filtered results
4. Export reports if needed

## Filtering Options

### Suspension Status
- **Suspended Clients**: View clients with suspended services
- **Active Clients**: View clients without suspensions
- **All Clients**: Include both suspended and active clients

### Date Filtering
- Filter by suspension date ranges
- Useful for periodic suspension analysis
- Track suspension trends over time

### Archive Status
- Include archived clients in reports
- Exclude archived clients for active client analysis
- Separate reporting for archived vs active clients

### Client Tags
- Filter by custom client tags
- Multiple tag selection support
- Useful for targeted client management
